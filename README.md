# ci-python-github-actions
- Displays a basic CI workflow using GitHub Actions.
- Workflow automatically runs Python script whenever code is updated on GitHub.

## Tech
- Python
- GitHub Actions

## Script 
- GitHub Actions is triggered on every update to the 'main' branch
- Workflow sets up Python
- Python script runs automatically with the CI process

### Purpose 
- Automate workflows
- Run scripts without manual execution
